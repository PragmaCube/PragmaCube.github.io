Syntaxe, variables, opérateurs, commentaires

Dans ce cours, nous verrons les principes de base de la programmation en C++. Après ce cours, vous serez capable de faire un programme récupérant des données via le clavier et affichant des données un peu traitées.

  • Accéder au cours

Structures conditionnelles et boucles

Dans ce cours, nous verrons comment répéter plusieurs opérations d'un coup. Nous verrons également comment traiter différentes conditions. Après ce cours, vous serez capable de programmer la plupart des algorithmes simples.

  • Accéder au cours

Tableaux, vecteurs et pointeurs

Dans ce cours, nous verrons comment stocker de grandes quantité de données en une seule variable. Nous aborderons aussi la notion d'adresse mémoire avec les pointeurs. Après ce cours, vous serez capable de mieux choisir le type de donnée en fonction de la situation.

  • Accéder au cours

Fonctions

Dans ce cours, nous verrons comment grouper plusieurs instructions en une seule. Après ce cours, vous serez capable de mieux architecturer votre programme.

  • Accéder au cours

Classes

Dans ce cours, nous verrons comment créer des types de données plus complexes pour des programmes de plus grande envergure. Après ce cours, vous serez capable de séparer votre code en plusieurs fichiers et utiliser des objets.

  • Accéder au cours

Espaces de noms, documentation et erreurs classiques

Dans ce cours, nous verrons comment regrouper du code en « section ». Nous survolerons également rapidement les erreurs de programmation les plus fréquentes. Après ce cours, vous serez capable de coder seul un programme moyennement complexe en C++.

  • Accéder au cours

Git et Github

Dans ce cours, nous verrons comment travailler à plusieurs sur un projet et comment versionner du coder. Après ce cours, vous serez capable d'utiliser Git et Github.

  • Accéder au cours

Composants électroniques

Dans ce cours, nous verrons les différents composants électroniques du robot et comment faire du câblage. Nous verrons également les différentes règles de sécurité lors de la manipulation de l'électronique. Après ce cours, vous serez en mesure de reconnaitre un composant sur le robot ou en installer un si nécessaire.

  • Accéder au cours

WPILIB et Command-Based Programming

Dans ce cours, nous verrons comment s'organise le code du robot ainsi que les méthodes les plus utilisés. Après ce cours, vous aurez une bonne idée du fonctionnement global du code et aurez les outils pour chercher dans les documentations.

  • Accéder au cours

Régulateur PID

Dans ce cours, nous verrons à quoi sert un régulateur PID et comment l'utiliser. Après ce cours, vous serez en mesure d'optimiser certaines actions du robot.

  • Accéder au cours

Concepts de base en programmation FRC

Dans ce cours, nous verrons certains concepts de base en programmation FRC comme le controle de moteurs, de capteurs et les interfaces homme / machine (joystick / XboxController).

  • Accéder au cours

Concepts avancés en programmation FRC

Dans ce cours, nous verrons quelques concepts avancés en programmation FRC, notamment la caméra, le couplage PID / feedforward et les Commands inline.
