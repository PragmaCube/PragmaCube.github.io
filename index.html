Syntaxe, variables, opérateurs, commentaires

Dans ce cours, nous verrons les principes de base de la programmation en C++. Après cette séance, vous serez capable de faire un programme récupérant des données via le clavier et affichant des données un peu traitées.

Structures conditionnelles et boucles

Dans ce cours, nous verrons comment répéter plusieurs opérations d'un coup. Nous verrons également comment traiter différentes conditions. Après cette séance, vous serez capable de programmer la plupart des algorithmes simples.

  • Accéder au cours

Tableaux, vecteurs et pointeurs

Dans ce cours, nous verrons comment stocker de grandes quantité de données en une seule variable. Nous aborderons aussi la notion d'adresse mémoire avec les pointeurs. Après cette séance, vous serez capable de mieux choisir le type de donnée en fonction de la situation.

  • Accéder au cours

Fonctions

Dans ce cours, nous verrons comment grouper plusieurs instructions en une seule. Après cette séance, vous serez capable de mieux architecturer votre programme.

  • Accéder au cours

Classes

Dans ce cours, nous verrons comment créer des types de données plus complexes pour des programmes de plus grande envergure. Après cette séance, vous serez capable de séparer votre code en plusieurs fichiers et utiliser des objets.

  • Accéder au cours

Espaces de noms, documentation et erreurs classiques

Dans ce cours, nous verrons comment regrouper du code en « section ». Nous survolerons également rapidement les erreurs de programmation les plus fréquentes. Après cette séance, vous serez capable de coder seul un programme moyennement complexe en C++.

  • Accéder au cours

Git et Github

Dans ce cours, nous verrons comment travailler à plusieurs sur un projet et comment versionner du coder. Après cette séance, vous serez capable d'utiliser Git et Github.
